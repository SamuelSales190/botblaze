# Strategy
      In _blaze_estrategy.csv  

      Example: X-10-P-V-B-1=P  
         X - Any number or color.  
         10 - number 10.  
         P -  color black  
         V - color red  
         B - color white  
         1 - number 1.  
         = - Separates signal from input.  
         P - Input color P, V or B.
